Настроить окружение с использованием инструментов виртуализации (docker, docker compose).
Проект должен содержать контейнеры: Golang, Nginx, MySQL и другие, нужные тебе для работы.

На основании задач, описанных в разделе “ОПИСАНИЕ ФУНКЦИОНАЛА СИСТЕМЫ” (ниже),
составить список задач, которые должны быть оформлены в виде таблицы, где есть:
- порядковый номер задачи,
- название задачи,
- подробное описание,
- временная оценка трудозатрат на эту задачу (например: "Реализация авторизации — 8 часов").

Создать проектную доску в Trello и перенести туда все задачи.
Доска должна состоять из следующих колонок:
- BACKLOG — все задачи,
- IN PROGRESS — задача над которой ты работаешь в данный момент,
- DONE —готовые задачи.

Спроектировать базу данных и написать миграции. 

1. Использовать MySQL в качестве базы данных.
2. Фронтенд часть реализовать c помощью Vue.js — https://vuejs.org/ .
3. Для реализации чата использовать веб-сокеты.
4. Рекомендуем использовать фреймворк Echo —  https://echo.labstack.com/  при  реализации серверной части на Go. Данный фреймворк также имеет функционал для работы с веб-сокетами  — https://echo.labstack.com/cookbook/websocket  и JWT  — https://echo.labstack.com/middleware/jwt.
